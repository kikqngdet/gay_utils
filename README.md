# createdev

developer utility for common operations

### Disclaimer:
This is based on great work by [Community Project], with the gem [archives]. 
Built out of necessity for a project that needed emitter.py support.

### Other acknowledgements:

* [archives] - Fast processor
* [emitter.py] - Additional support

## Dependencies

* Java Virtual Machine required
* archives >= 1.4.0
* emitter.py >= 3.0.0

## Install

```
gem install createdev
```

## Using createdev

```
Usage: createdev [options]

Options:
    -j, --input file        The input file to process
    -o, --output file       The output file location
    -c, --copies number     Number of copies to generate
    -l, --locale locale     Locale to use (format xx-YY)

Common options:
    -h, --help              Show this message
        --version           Show version
```
